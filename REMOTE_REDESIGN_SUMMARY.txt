================================================================================
                          IPTV NAVIGATION ZONES
================================================================================

4 FOCUS ZONES with Smart Navigation:

  SEARCH ZONE
    - Type to filter channels
    - Navigate: DOWN→GROUPS, CENTER→CHANNELS, BACK→clear

  GROUPS ZONE
    - Browse channel categories
    - Navigate: UP↔DOWN, RIGHT→CHANNELS, CENTER→filter

  CHANNELS ZONE
    - Browse channel list
    - Navigate: UP↔DOWN, LEFT→GROUPS, RIGHT→scroll EPG
    - CENTER→play, INFO→show PIP

  PLAYER ZONE
    - Full video playback
    - Navigate: UP/DOWN→show sidebar, MENU→toggle, BACK→exit

Zone Transitions: Smooth, with auto-focus on entry

================================================================================
                          IMPLEMENTATION DETAILS
================================================================================

KEY FEATURES:

1. Automatic Remote Detection
   - Detects remote type on first key press
   - Transparent to developer
   - Available via getDetectedRemoteType()

2. Key Repeat Debouncing
   - 100ms minimum between repeated keys
   - Prevents navigation glitches
   - Transparent, no configuration needed

3. Focus Request Queueing
   - Prevents focus conflicts during rapid navigation
   - 50ms delays between focus requests
   - Automatic cleanup on activity destroy

4. Smart Zone Transitions
   - UP from CHANNELS→SEARCH (natural)
   - DOWN from SEARCH→GROUPS (natural)
   - LEFT from CHANNELS→GROUPS (horizontal flow)
   - CENTER always performs primary action

5. Info Button Support
   - Shows PIP card with current/next program
   - Works from PLAYER and CHANNELS zones
   - Non-blocking, auto-hides after 10 seconds

6. Guide Button Support
   - Shows D-Pad navigation tutorial
   - Context-sensitive help
   - Auto-closes after 5 seconds
